# キャラクター電卓 (Character Calculator)

好きなキャラクターが接客してくれる、カスタマイズ可能なWEB電卓アプリです。
AIイベントフェスティバル展示作品。

## 🌟 コンセプト
「お気に入りのキャラクターに合計金額を伝えてもらいたい！」
そんな想いを形にしました。背景画像を差し替えるだけで、世界に一つだけの「推し電卓」に変身します。

## ✨ 特徴
- **2つのモード搭載**:
  - **購入モード**: 商品の金額を次々加算し、合計を表示します。
  - **おつりモード**: 預かり金額からお会計を引いたおつりを計算します。
- **簡単カスタマイズ**: `dentaku.png` を自分の好きなキャラ画像（透過PNG推奨）に置き換えるだけで、接客キャラクターを変更可能。
- **サイバーなUI**: ダークモードを基調とした、直感的に操作できるデザイン。

## 🚀 使い方
1. 本リポジトリをクローンまたはダウンロードします。
2. `index.html` をブラウザで開くと起動します。
3. **キャラの変え方**: 
   - `dentaku.png` と同じファイル名で、お好みの画像をプロジェクトフォルダに保存してください。
   - 推奨設定：背景が透明な透過PNG形式。

## 🛠 技術スタック
- HTML5 / CSS3 (Grid Layout, CSS Animation)
- JavaScript (Vanilla JS)

✨ 魅せる接客UI (Interactive Visuals)
- ネオンエフェクト: ボタンを押すとサイバーパンクなネオンライトが走るアニメーションを搭載。単調になりがちな会計待ちの時間を、エンターテインメントに変えます。
- 対面レイアウト: タブレットを平置きしたり、お客さん側に向けたりすることを想定した反転表示や、没入感のあるフルスクリーンデザインを採用しています。

## 📱 推奨環境
- タブレット（横向き）
- PCブラウザ
※レスポンシブ対応ですが、横画面での利用に最適化されています。
